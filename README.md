# 🦟 Dengue Awareness Website (3D)

This is a 3D educational website about **Dengue Fever**, built with:
- HTML
- CSS
- JavaScript (Three.js)

## 🚀 Features
- 3D animated dengue virus model
- Sections: About, Symptoms, Prevention, Treatment, Statistics
- Responsive design
- Hover 3D effects
